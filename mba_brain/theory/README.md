# Theory Directory — PhD-Level Academic Foundations

This directory contains formal academic theory underlying the MBA Brain's practical modules. Each file provides rigorous theoretical foundations with formal citations, mathematical derivations, and connections to the operator-level content in the numbered modules.

---

## Purpose

The MBA Brain's numbered modules (00-19) provide **operator-level** knowledge — practical, actionable, and oriented toward decision-making. This theory directory provides the **academic foundations** that underpin those modules, including:

- Formal mathematical derivations
- Peer-reviewed citations (APA format)
- Proof sketches for key theorems
- Connections between theory and practice

## Citation Convention

All citations follow APA 7th edition format. Key works are cited as:

```
Author, A. A. (Year). Title of work. Publisher.
Author, A. A., & Author, B. B. (Year). Title of article. Journal Name, Volume(Issue), pages.
```

Mathematical notation follows standard economics conventions:
- Vectors in bold: **x**
- Sets in calligraphic: U, P
- Derivatives: f'(x), df/dx, partial f / partial x
- Expectations: E[X]
- Probability: P(A), P(A|B)
