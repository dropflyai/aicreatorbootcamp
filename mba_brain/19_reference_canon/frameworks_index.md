## How to Use This Index

1. **Identify the decision type** — strategy, financial, product, operations, people
2. **Scan relevant frameworks** — select 2-3 that apply
3. **Apply frameworks** — use the detailed treatment in the referenced module
4. **Triangulate** — when multiple frameworks agree, confidence increases; when they disagree, investigate the source of disagreement
