## Key Decision Frameworks

### Hiring Decision Tree
```
Need identified → Headcount approved? → Internal candidate? → External search
                                      → Yes → Internal mobility process
                  → No → Build business case → ROI analysis → Re-submit
```

### Compensation Decision
```
Role defined → Market data pulled → Internal equity check → Band placement
            → Offer constructed → Total rewards calculated → Approval chain
```

### Performance Intervention
```
Issue identified → Documentation review → Coaching conversation
                → Improvement? → Yes → Monitor and recognize
                → No → PIP → Improvement? → Yes → Return to normal
                                           → No → Separation process
```
