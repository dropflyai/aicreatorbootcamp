# Stochastic Calculus for Options Trading

**Prerequisite:** Measure theory, probability theory, real analysis
**Relevance:** Foundation for all continuous-time options pricing models

---

## 1. Probability Spaces and Filtrations

### Formal Setup

A filtered probability space (Omega, F, {F_t}_{t>=0}, P) consists of:

- **Omega:** Sample space of all possible market outcomes (paths)
- **F:** Sigma-algebra of measurable events
- **{F_t}:** Filtration representing information available at time t (natural filtration of price process)
- **P:** Physical (real-world) probability measure

The filtration satisfies F_s subset F_t for s < t (information increases over time). In finance, F_t is typically generated by observed price paths up to time t.

### Adapted Processes

A stochastic process X_t is adapted to {F_t} if X_t is F_t-measurable for all t. This means the value of the process at time t is determined by information available at time t. All tradable asset prices must be adapted processes.

## 2. Brownian Motion (Wiener Process)

### Definition

A standard Brownian motion W_t is a continuous-time stochastic process satisfying:

1. **W_0 = 0** (starts at origin)
2. **Independent increments:** W_t - W_s is independent of F_s for t > s
3. **Normal increments:** W_t - W_s ~ N(0, t - s) for t > s
4. **Continuous paths:** t -> W_t is continuous almost surely

### Key Properties

- **E[W_t] = 0** for all t (zero drift)
- **Var(W_t) = t** (variance grows linearly with time)
- **Cov(W_s, W_t) = min(s, t)** (covariance structure)
- **Quadratic variation:** [W, W]_t = t (accumulates variance at rate 1)
- **Nowhere differentiable:** Paths are continuous but have no derivative at any point
- **Self-similarity:** For any c > 0, the process c^{-1/2} W_{ct} is also a standard Brownian motion
- **Markov property:** Future evolution depends only on current value, not history
- **Martingale property:** E[W_t | F_s] = W_s for s <= t

### Quadratic Variation (Critical for Ito's Lemma)

For a partition 0 = t_0 < t_1 < ... < t_n = T:

```
[W, W]_T = lim_{n->inf} sum_{i=0}^{n-1} (W_{t_{i+1}} - W_{t_i})^2 = T
```

This non-zero quadratic variation distinguishes stochastic calculus from ordinary calculus. In ordinary calculus, smooth functions have zero quadratic variation, so the (dW)^2 term vanishes. In stochastic calculus, (dW_t)^2 = dt, which gives rise to the extra term in Ito's Lemma.

### Computational Rules (Ito Multiplication Table)

```
dt * dt   = 0
dt * dW_t = 0
dW_t * dt = 0
dW_t * dW_t = dt
```

These rules are the engine of all stochastic calculus computations.

## 3. Ito Integral

### Construction

The Ito integral integral_0^T f(t, omega) dW_t is constructed as a limit of sums:

```
I_n = sum_{i=0}^{n-1} f(t_i, omega) * (W_{t_{i+1}} - W_{t_i})
```

Key distinction from Stratonovich: the integrand is evaluated at the LEFT endpoint t_i (not the midpoint). This ensures the integral is a martingale.

### Ito Isometry

For adapted process f with E[integral_0^T f^2 dt] < infinity:

```
E[(integral_0^T f dW_t)^2] = E[integral_0^T f^2 dt]
```

This connects L^2 norms and is fundamental for computing variances.

### Properties

- **E[integral_0^T f dW_t] = 0** (Ito integrals have zero expectation)
- **Martingale:** The process M_t = integral_0^t f dW_s is a martingale
- **Linearity:** integral (af + bg) dW = a integral f dW + b integral g dW

## 4. Ito's Lemma (The Chain Rule of Stochastic Calculus)

### Statement

Let X_t satisfy dX_t = mu(t, X_t) dt + sigma(t, X_t) dW_t and let f(t, x) be twice continuously differentiable. Then:

```
df(t, X_t) = [df/dt + mu * df/dx + (1/2) * sigma^2 * d^2f/dx^2] dt + sigma * df/dx * dW_t
```

### Derivation (Intuition)

Start with Taylor expansion to second order:

```
df = (df/dt) dt + (df/dx) dX + (1/2)(d^2f/dx^2)(dX)^2 + (1/2)(d^2f/dt^2)(dt)^2 + (d^2f/dtdx)(dt)(dX) + ...
```

Substitute dX = mu dt + sigma dW:

```
(dX)^2 = mu^2 (dt)^2 + 2*mu*sigma*(dt)(dW) + sigma^2 (dW)^2
```

Apply Ito multiplication rules: (dt)^2 = 0, (dt)(dW) = 0, (dW)^2 = dt:

```
(dX)^2 = sigma^2 dt
```

The second-order term survives because of the non-zero quadratic variation of Brownian motion. This is the fundamental difference from ordinary calculus.

### Multidimensional Ito's Lemma

For f(t, X_1, ..., X_n) where each X_i follows an Ito process:

```
df = (df/dt) dt + sum_i (df/dx_i) dX_i + (1/2) sum_{i,j} (d^2f/dx_i dx_j) dX_i dX_j
```

With dW_i dW_j = rho_{ij} dt for correlated Brownian motions.

## 5. Geometric Brownian Motion (GBM)

### The Model

The Black-Scholes asset price model:

```
dS_t = mu * S_t * dt + sigma * S_t * dW_t
```

where mu is the drift (expected return) and sigma is the volatility.

### Solution via Ito's Lemma

Apply Ito's Lemma to f(S) = ln(S):

```
d(ln S) = (1/S) dS - (1/2)(1/S^2)(dS)^2
         = (1/S)(mu*S dt + sigma*S dW) - (1/2)(1/S^2)(sigma^2*S^2 dt)
         = (mu - sigma^2/2) dt + sigma dW_t
```

Integrating both sides from 0 to T:

```
ln(S_T) - ln(S_0) = (mu - sigma^2/2)*T + sigma*W_T
```

Therefore:

```
S_T = S_0 * exp((mu - sigma^2/2)*T + sigma*W_T)
```

### Properties of GBM

- S_T is log-normally distributed: ln(S_T/S_0) ~ N((mu - sigma^2/2)T, sigma^2 T)
- **E[S_T] = S_0 * exp(mu * T)** (expected value grows at rate mu)
- **Var(S_T) = S_0^2 * exp(2*mu*T) * (exp(sigma^2 * T) - 1)**
- Prices are always positive (exp never crosses zero)
- The sigma^2/2 correction is the "convexity adjustment" -- the median of S_T is less than the mean

### Limitations

- Constant volatility (empirically violated -- see volatility smile)
- Continuous paths (no jumps -- see Merton jump-diffusion)
- Log-normal returns (fat tails in reality)
- Independent increments (volatility clustering observed empirically)

## 6. Stochastic Differential Equations (SDEs)

### General Form

```
dX_t = mu(t, X_t) dt + sigma(t, X_t) dW_t
```

### Existence and Uniqueness (Ito Conditions)

Under Lipschitz and linear growth conditions:

```
|mu(t,x) - mu(t,y)| + |sigma(t,x) - sigma(t,y)| <= K|x - y|
|mu(t,x)|^2 + |sigma(t,x)|^2 <= K^2(1 + |x|^2)
```

there exists a unique strong solution.

### Important SDEs in Finance

| Model | SDE | Application |
|-------|-----|-------------|
| GBM | dS = mu*S dt + sigma*S dW | Stock prices |
| Ornstein-Uhlenbeck | dX = theta*(mu - X) dt + sigma dW | Mean-reverting rates |
| CIR (Cox-Ingersoll-Ross) | dr = kappa*(theta - r) dt + sigma*sqrt(r) dW | Interest rates |
| Heston | dv = kappa*(theta - v) dt + xi*sqrt(v) dW | Stochastic volatility |
| CEV | dS = mu*S dt + sigma*S^beta dW | Leverage effect |

## 7. Girsanov's Theorem

### Statement

Let W_t be a Brownian motion under measure P. Define the Radon-Nikodym derivative:

```
dQ/dP = Z_T = exp(-integral_0^T theta_t dW_t - (1/2) integral_0^T theta_t^2 dt)
```

where theta_t is an adapted process satisfying the Novikov condition: E[exp((1/2) integral_0^T theta_t^2 dt)] < infinity.

Then under the new measure Q:

```
W_t^Q = W_t + integral_0^t theta_s ds
```

is a standard Brownian motion.

### Financial Interpretation

Under the physical measure P, the stock price follows:

```
dS/S = mu dt + sigma dW_t^P
```

Set theta = (mu - r) / sigma (the market price of risk, or Sharpe ratio). By Girsanov, under the risk-neutral measure Q:

```
dS/S = r dt + sigma dW_t^Q
```

All assets earn the risk-free rate under Q. This is the mathematical foundation of risk-neutral pricing. The real-world drift mu is irrelevant for option pricing -- only volatility sigma matters.

### Novikov Condition

The condition E[exp((1/2) integral_0^T theta_t^2 dt)] < infinity ensures Z_T is a true martingale (not just a local martingale), guaranteeing the measure change is valid.

## 8. Risk-Neutral Pricing and Martingale Measures

### Fundamental Theorem of Asset Pricing (First)

A market is arbitrage-free if and only if there exists at least one equivalent martingale measure Q under which all discounted asset prices are martingales.

### Fundamental Theorem of Asset Pricing (Second)

The market is complete (every contingent claim is replicable) if and only if the equivalent martingale measure Q is unique.

### Risk-Neutral Pricing Formula

The price at time t of a derivative with payoff H at time T:

```
V_t = e^{-r(T-t)} * E^Q[H | F_t]
```

This expectation is taken under the risk-neutral measure Q, not the physical measure P.

### Why It Works

Under Q, the discounted portfolio value e^{-rt} V_t is a martingale. By the martingale property:

```
e^{-rt} V_t = E^Q[e^{-rT} H | F_t]
```

This connects the derivative price to a conditional expectation, which can be computed via:
- Analytic formulas (Black-Scholes for European options)
- Monte Carlo simulation (simulate paths under Q)
- PDE methods (Feynman-Kac connection)

## 9. Martingale Representation Theorem

### Statement

If M_t is a martingale adapted to the filtration generated by a Brownian motion W_t, then there exists a unique adapted process phi_t such that:

```
M_t = M_0 + integral_0^t phi_s dW_s
```

### Financial Implication: Market Completeness

Every derivative payoff H can be replicated by a self-financing portfolio. The hedging strategy (number of shares of the underlying to hold) is exactly phi_t. This is why the Black-Scholes delta hedge works -- the martingale representation theorem guarantees the existence of a replicating portfolio.

### Connection to Delta Hedging

For a European call option C(t, S_t):

```
dC = (dC/dS) dS + [...] dt = Delta * dS + [...]dt
```

The replicating portfolio holds Delta shares of stock. The martingale representation theorem tells us this hedge is exact in the continuous-time limit.

## 10. Feynman-Kac Formula

### Statement

Consider the PDE:

```
dv/dt + mu(t,x) * dv/dx + (1/2) * sigma(t,x)^2 * d^2v/dx^2 - r*v = 0
```

with terminal condition v(T, x) = g(x).

The solution is:

```
v(t, x) = E^Q[e^{-r(T-t)} * g(X_T) | X_t = x]
```

where X_t satisfies dX_t = mu dt + sigma dW_t under Q.

### Financial Interpretation

The Feynman-Kac formula establishes the deep equivalence between:

1. **PDE approach:** Solve the Black-Scholes PDE to find option prices
2. **Probabilistic approach:** Compute risk-neutral expectations

The Black-Scholes PDE and the risk-neutral expectation formula give identical answers. This duality is exploited computationally:
- Use PDE methods (finite differences) for low-dimensional problems
- Use Monte Carlo (probabilistic) for high-dimensional problems

## 11. Applications to Options Trading Systems

### Model Calibration Pipeline

```
Market Data -> Implied Vol Surface -> Choose SDE Model -> Calibrate Parameters -> Price Exotics
```

The choice of SDE directly impacts:
- Delta hedging frequency and accuracy
- Exotic option pricing
- Risk metric computation (VaR, Greeks)

### Key Design Decisions

1. **GBM vs. Local Vol vs. Stochastic Vol:** GBM is the baseline; local vol matches the smile exactly but has unrealistic forward dynamics; stochastic vol (Heston) captures vol-of-vol
2. **Measure choice:** Always price under Q, simulate P&L paths under P for risk management
3. **Discretization scheme:** Euler-Maruyama for simple SDEs, Milstein for better convergence, exact simulation for GBM and CIR

### Numerical Methods

| Method | When to Use | Convergence |
|--------|-------------|-------------|
| Euler-Maruyama | General SDEs | O(sqrt(dt)) weak, O(sqrt(dt)) strong |
| Milstein | When d(sigma)/dX is tractable | O(dt) strong |
| Exact simulation | GBM, OU, CIR | Exact (no discretization error) |
| Log-Euler | GBM specifically | Preserves positivity |

---

## 12. Common Pitfalls

- **Confusing P and Q measures:** Drift mu is for P (real-world forecasting); drift r is for Q (pricing)
- **Ignoring convexity adjustment:** E[f(X)] != f(E[X]) by Jensen's inequality; the sigma^2/2 term in GBM matters
- **Assuming zero quadratic variation:** The (dW)^2 = dt rule is non-negotiable; dropping it gives wrong answers
- **Novikov condition violations:** Explosive processes can make the Girsanov measure change fail
- **Discretization bias:** Euler schemes on GBM can produce negative prices; use log-Euler or exact simulation
