> Layout and copy for a single printed page (8.5" x 11") or digital flyer.

## LAYOUT NOTES

**Suggested format:** Two-column layout with a bold header spanning full width at the top and a footer strip at the bottom. Print on white or light gray stock. Use a strong accent color (electric green or deep blue) for headlines and callout boxes.

## DESIGN DIRECTION SUMMARY

| Element | Recommendation |
|---|---|
| Background | White body, dark header and footer strips |
| Accent color | Electric green (#BFFF00) or strong blue (#2563EB) |
| Headline font | Heavy/Black weight, tight leading |
| Body font | Regular weight, 10–11pt for print |
| Imagery | Abstract AI-generated visuals or tool screenshots as background texture |
| Format | Portrait 8.5" x 11" for print; 1080x1920px for digital/Instagram Stories |
