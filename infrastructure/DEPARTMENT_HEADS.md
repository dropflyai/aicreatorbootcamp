Marketing ←→ Sales ←→ Customer Success (Revenue triad)
Finance ←→ Legal ←→ Operations (Admin triad)

Security ←→ Engineering ←→ Cloud (Infrastructure triad)

## Invocation Pattern

To invoke a Department Head with their specialists:

```typescript
await Task({
  description: "Engineering Department: Build auth system",
  prompt: `
    You are the VP of Engineering (engineering_brain).

    TASK: Build user authentication

    As Department Head, you must:
    1. Assess task complexity
    2. Decide which specialists to spawn
    3. Coordinate their work
    4. Synthesize outputs
    5. Report results

    Available specialists:
    - Frontend Developer
    - Backend Developer
    - Database Engineer
    - Security Engineer

    Spawn specialists as parallel Task agents.
    Follow engineering_brain protocols.
  `,
  subagent_type: "general-purpose"
});
```
