# Venture Building -- Corporate New Venture Creation

## Overview

Venture building is the disciplined process of creating new businesses from scratch
within or alongside an established organization. Unlike incremental innovation that
improves existing products, venture building creates entirely new value propositions,
business models, and potentially new markets. This module covers the full spectrum of
corporate venture building approaches, from internal startups to venture studios to
spin-outs, providing the frameworks needed to build new businesses systematically.

---

## Internal Startup Model

### Definition

An internal startup is a small, autonomous team within a larger organization tasked
with building a new business from zero. It operates with startup-like speed and
autonomy while leveraging corporate assets.

### Internal Startup Structure

```
+------------------------------------------------------------------+
|                     PARENT CORPORATION                             |
|                                                                  |
|  Core Business Units        Internal Startup                     |
|  +------------------+       +------------------------+           |
|  | Business Unit A  |       | Startup Team           |           |
|  | Business Unit B  |       | - GM / Entrepreneur    |           |
|  | Business Unit C  |       | - Product              |           |
|  |                  |       | - Engineering           |           |
|  |                  |       | - Design                |           |
|  |  (Exploit)       |       | - Growth                |           |
|  +------------------+       | (Explore)               |           |
|         |                   +------------------------+           |
|         |                          |                              |
|         +--- Shared services ------+                              |
|         |    (Legal, HR, Finance, IT, Brand)                     |
|         |                          |                              |
|         +--- Corporate assets -----+                              |
|              (Distribution, Data, Customers, Brand)              |
+------------------------------------------------------------------+
```

### Internal Startup Operating Principles

| Principle | Implementation |
|-----------|---------------|
| Autonomy | Separate reporting, own budget, own metrics, own culture |
| Speed | 2-week sprints, direct customer access, minimal approvals |
| Accountability | Clear milestones, stage-gate funding, kill criteria |
| Access to assets | Can leverage corporate distribution, data, brand (opt-in) |
| Protection | Shielded from corporate processes, politics, "antibodies" |
| Executive sponsor | Senior leader who protects and advocates for the venture |

### Common Failure Modes

| Failure Mode | Cause | Prevention |
|-------------|-------|-----------|
| Antibody rejection | Corporate immune system kills innovation | Executive sponsor protection |
| Resource starvation | Core business absorbs budget during downturns | Ring-fenced funding |
| Metric mismatch | Judged by core business metrics (revenue, margin) | Innovation accounting metrics |
| Talent drain | Best people pulled back to core business | Dedicated team, career path |
| Scope creep | Corporate requests expand scope beyond startup capacity | Clear charter and boundaries |
| Culture clash | Startup culture clashes with corporate culture | Physical and organizational separation |

## Venture Studio Model

### Definition

A venture studio (also called startup studio, venture builder, or startup factory)
is a dedicated organization that systematically creates new companies using a
repeatable methodology.

### Venture Studio Process

```
IDEATION        VALIDATION      BUILDING        SCALING         EXIT/INTEGRATE
(0-3 months)    (3-6 months)    (6-18 months)   (18-36 months)  (36+ months)

Generate        Test problem    Build MVP,      Scale go-to-    IPO, acquisition,
ideas from      and solution    find PMF,       market, raise   or integrate
thesis, market  with lean       build team,     external        into parent
research,       experiments     raise seed      funding         corporation
customer pain                   funding

Kill rate:      Kill rate:      Kill rate:      Kill rate:      Success rate:
~70%            ~50%            ~30%            ~20%            ~10-15% of
                                                                initial ideas
```

### Venture Studio Economics

```
Portfolio approach (example):

Start 10 ventures per year
  --> 3 survive validation (70% kill rate)
  --> 2 achieve product-market fit
  --> 1 scales to significant revenue
  --> Expected value per cohort: 1 successful business

Investment per venture:
  Ideation phase: $50K-100K
  Validation phase: $100K-300K
  Building phase: $500K-2M
  Total portfolio investment per cohort: $3-10M

Expected returns:
  1 successful business valued at $20-100M+
  Portfolio IRR target: 25-40%
```

### Venture Studio vs. Accelerator vs. Incubator

| Dimension | Venture Studio | Accelerator | Incubator |
|-----------|---------------|-------------|-----------|
| Who has the idea | Studio generates ideas | Founders bring ideas | Founders bring ideas |
| Equity ownership | 30-80% (studio built it) | 5-10% | 0-5% |
| Team | Studio assigns/recruits | Founders have team | Founders have team |
| Duration | 18-36 months | 3-6 months (cohort) | 6-24 months (flexible) |
| Services | Full operational support | Mentorship, demo day | Space, basic support |
| Funding | Studio funds initially | Small investment | None or small |
| Repeatable | Systematic methodology | Cohort model | Ad hoc |

## Spin-Out vs. Spin-In

### Spin-Out (Inside to Outside)

```
PARENT CORPORATION                   INDEPENDENT COMPANY
+-------------------+                +-------------------+
|                   |    Spin-Out    |                   |
|  Internal Venture |  ---------->  |  New Company      |
|  (proved PMF,     |    Transfer:  |  (own board,      |
|   needs freedom   |    - Team     |   own funding,    |
|   to scale)       |    - IP       |   own culture)    |
|                   |    - Assets   |                   |
+-------------------+               +-------------------+
        |                                   |
        +--- Parent retains equity stake ---+
        +--- Possible commercial agreement -+
```

### When to Spin Out

| Signal | Rationale |
|--------|-----------|
| The venture needs a different culture to scale | Corporate culture constrains growth |
| The venture needs external funding | Corporate capital allocation process too slow |
| The venture is in a different market | Brand confusion or channel conflict |
| Talent retention requires equity incentives | Corporate compensation does not attract startup talent |
| Speed requires independence | Corporate governance slows decision-making |

### Spin-In (Outside to Inside)

The reverse: acquiring an external startup or project and integrating it into
the corporate structure. Common in acqui-hires and technology acquisitions.

## Funding Mechanisms

### Internal Venture Funding Models

| Model | How It Works | Pros | Cons |
|-------|-------------|------|------|
| Internal VC fund | Dedicated fund, VC-like process | Professional, disciplined | May lack patient capital |
| Dedicated budget | Annual innovation budget allocation | Predictable, simple | Vulnerable to cuts |
| Milestone-based | Funding released at stage gates | Disciplined, reduces waste | Can create perverse incentives |
| Business unit sponsorship | BU funds ventures in their domain | Strategic alignment | Limits scope to BU interests |
| CEO discretionary | CEO allocates from discretionary budget | Fast, flexible | Single point of failure |
| Innovation tax | Each BU contributes % of revenue | Sustainable, distributed | Feels like a tax; resentment |

### Milestone-Based Funding Structure

```
Stage           Funding         Duration     Key Milestone
------          -------         --------     ----------------
Discovery       $50-100K        2-3 months   Problem validated (30+ interviews)
Validation      $100-300K       3-6 months   Solution validated (MVP + early users)
Efficiency      $300K-1M        6-12 months  Unit economics demonstrated
Scale           $1-5M           12-24 months Revenue growth + retention proven
Graduation      $5-20M+         24+ months   Standalone P&L or spin-out

Each stage requires a formal stage-gate review.
Funding is NOT guaranteed -- ventures can be killed at any gate.
```

## Team Composition

### Venture Team Archetypes

| Role | Responsibility | Archetype |
|------|---------------|-----------|
| Venture Lead / GM | Overall venture ownership, strategy, P&L | Entrepreneur |
| Product Lead | Customer discovery, product vision, roadmap | Product manager with startup DNA |
| Technical Lead | Architecture, engineering, technical feasibility | Full-stack builder |
| Growth Lead | Customer acquisition, marketing, distribution | Growth hacker |
| Design Lead | UX research, prototyping, user experience | Designer who codes |

### Entrepreneur-in-Residence (EIR)

| Aspect | Detail |
|--------|--------|
| Definition | Experienced entrepreneur embedded in the organization to lead ventures |
| Tenure | Typically 6-18 months |
| Compensation | Base salary + equity/carry in ventures they lead |
| Reporting | To Chief Innovation Officer or VP Innovation |
| Value | Brings startup mindset, speed, risk tolerance, fundraising experience |
| Risk | Cultural clash, short tenure limits impact, expensive |

### Team Size by Stage

```
Discovery (2-3 people):
  Venture Lead + 1 Engineer + 1 Designer

Validation (3-5 people):
  + Growth Lead + 1 more Engineer

Building (5-10 people):
  + Product Lead + Engineers + Customer Success

Scaling (10-25 people):
  + Full functional team (Sales, Marketing, Operations)

Graduation (25+ people):
  Full company structure
```

## Stage Gates for Ventures

### Gate Criteria

| Gate | Key Questions | Evidence Required | Kill Criteria |
|------|--------------|-------------------|--------------|
| Gate 0: Idea | Is this worth exploring? | Innovation thesis alignment, market sizing | No strategic fit |
| Gate 1: Problem | Is this a real problem? | 30+ customer interviews, pain validated | < 50% confirm problem |
| Gate 2: Solution | Does our solution work? | MVP tested, early users engaged | No user engagement |
| Gate 3: Model | Do the economics work? | Unit economics, retention data | LTV:CAC < 1:1 |
| Gate 4: Scale | Can we grow? | Growth channels identified, early traction | No scalable channel |
| Gate 5: Graduate | Is this a real business? | Revenue, team, market position | Cannot stand alone |

## Graduation Criteria

### When a Venture "Graduates"

A venture graduates from the innovation portfolio to become a standing business
when it meets these criteria:

```
GRADUATION SCORECARD

Revenue:
  [ ] Monthly recurring revenue > $____/month
  [ ] Revenue growing > ____% month-over-month
  [ ] 3+ months of consistent growth

Unit Economics:
  [ ] LTV:CAC ratio > 3:1
  [ ] Gross margin > 60%
  [ ] Payback period < 12 months

Product:
  [ ] Product-market fit confirmed (Sean Ellis > 40%)
  [ ] Retention curve flattens (not declining to zero)
  [ ] Core feature set stable

Team:
  [ ] Full leadership team in place
  [ ] Team can operate independently
  [ ] Culture and values established

Market:
  [ ] Clear market positioning
  [ ] Defensible competitive advantage
  [ ] Addressable market > $____
```

## Key Takeaways

1. **Structure for speed and protection**: Internal ventures need autonomy AND corporate assets.
2. **Fund in stages**: Milestone-based funding prevents over-investment in losers.
3. **Kill early, kill often**: 70%+ of ventures should be killed before significant investment.
4. **Team is everything**: An EIR or venture lead with startup experience is essential.
5. **Graduation criteria must be pre-defined**: Prevents political capture of the innovation portfolio.
6. **Spin-out when corporate constraints limit growth**: Independence enables scale.
