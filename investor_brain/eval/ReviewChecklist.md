# Investor Review Checklist -- Authoritative

Use this checklist to evaluate any investor-facing deliverable, fundraising process, or investor relations activity.
Every section must be completed. Incomplete reviews are invalid.

If an investor deliverable cannot pass this checklist, it is not ready for the market.
