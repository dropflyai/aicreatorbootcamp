# Operating Cadence -- The CEO's Rhythm of Execution

## Overview

Operating cadence is the structured rhythm of meetings, reviews, decisions,
and communications that defines how an organization operates day-to-day,
week-to-week, and quarter-to-quarter. It is the operating system of
organizational execution.

Without cadence, organizations operate reactively -- lurching from crisis to
crisis, making decisions when they become urgent rather than when they are
important. With cadence, organizations operate proactively -- making decisions
at the right time, reviewing performance at regular intervals, and
communicating consistently.

Keith Rabois (PayPal, LinkedIn, Square) describes cadence as "the heartbeat
of the company." If the heartbeat stops, the company dies.

## The CEO's Daily Cadence

### Daily Rhythm (Typical CEO Day)

```
TIME         ACTIVITY                    PURPOSE
--------     --------------------------  --------------------------------
7:00-8:00    Personal time               Energy management, exercise,
                                         reading, reflection
8:00-8:30    Dashboard review            30-second health check: metrics,
                                         alerts, overnight developments
8:30-9:00    Priority setting            Top 3 things for today
                                         (first things first -- Drucker)
9:00-12:00   Deep work block             Type 1 decisions, strategy,
                                         writing, thinking
12:00-1:00   Lunch meeting               Relationship building (internal
                                         or external)
1:00-4:00    Meetings block              1:1s, team meetings, external
4:00-5:00    Decision hour               Pending decisions that need
                                         CEO input
5:00-5:30    Day close                   Review actions, prepare tomorrow
```

### Daily Decision Practices

1. **Morning triage** -- Review all items requiring CEO input. Classify:
   - Decide now (Type 2, enough information)
   - Schedule decision (Type 1, need more information)
   - Delegate (does not require CEO)
   - Defer (not time-sensitive, add to backlog)

2. **The 3-3-3 Method** -- Each day, the CEO completes:
   - 3 hours of deep work on the most important strategic priority
   - 3 short tasks (emails, approvals, quick decisions)
   - 3 maintenance activities (1:1s, reviews, communications)

### Daily CEO Brain Operations

```
DAILY CEO BRAIN CHECKLIST
- [ ] Metrics dashboard reviewed (growing? healthy? on track?)
- [ ] Critical blockers identified and owned
- [ ] At least one Type 1 decision advanced
- [ ] At least one strategic priority progressed
- [ ] Direct reports have what they need for today
- [ ] No decision has been waiting more than 48 hours
```

## The CEO's Weekly Cadence

### Weekly Rhythm

```
MONDAY: ORIENTATION
+--------------------------------------------------+
| Morning: Leadership team standup (30 min)        |
|   - Each leader: top priority, biggest blocker   |
|   - CEO: strategic context for the week          |
| Afternoon: Individual 1:1s (direct reports)      |
+--------------------------------------------------+

TUESDAY-WEDNESDAY: EXECUTION
+--------------------------------------------------+
| Deep work, external meetings, customer/partner   |
| time. CEO available for escalations.             |
+--------------------------------------------------+

THURSDAY: REVIEW
+--------------------------------------------------+
| Morning: Weekly Business Review (60 min)         |
|   - Metrics review                               |
|   - Red flag discussion                          |
|   - Deep dive on one topic                       |
| Afternoon: Cross-functional coordination         |
+--------------------------------------------------+

FRIDAY: REFLECTION AND COMMUNICATION
+--------------------------------------------------+
| Morning: Strategy time (thinking, writing)       |
| Afternoon: CEO weekly update (written, company)  |
|   - What happened this week                      |
|   - What we learned                              |
|   - What to focus on next week                   |
| End of day: Personal retrospective               |
+--------------------------------------------------+
```

### Weekly 1:1 Design

The 1:1 is the CEO's most important recurring meeting (Grove). Structure:

```
1:1 MEETING (30-60 min, weekly)
+--------------------------------------------------+
| THEIR AGENDA (first 20 min):                     |
| - Their top priorities and blockers              |
| - Issues they want CEO input on                  |
| - Team health and dynamics                       |
|                                                  |
| CEO AGENDA (next 15 min):                        |
| - Strategic context and upcoming changes         |
| - Feedback (positive and constructive)           |
| - Cross-functional information sharing           |
|                                                  |
| COACHING (last 10 min):                          |
| - Development areas                              |
| - Career growth                                  |
| - Support needed                                 |
+--------------------------------------------------+

RULES:
- NEVER cancel a 1:1. Reschedule if necessary.
- The direct report owns the agenda.
- CEO listens more than talks (70/30 rule).
- Action items are captured and tracked.
```

Threshold system:
- **Green:** On track (within 10% of target)
- **Yellow:** At risk (10-25% below target)
- **Red:** Off track (>25% below target or trending wrong)

Every RED metric gets an owner and an action plan by end of day.

## The CEO's Monthly Cadence

### Monthly Rhythm

```
FIRST WEEK: REVIEW
+--------------------------------------------------+
| Monthly business review (2 hours)                |
|   - Full financial review                        |
|   - OKR progress (on track / at risk / off track)|
|   - Customer health review                       |
|   - Team health review                           |
+--------------------------------------------------+

SECOND WEEK: ALL-HANDS
+--------------------------------------------------+
| All-hands meeting (60 min)                       |
|   - CEO strategic update                         |
|   - Key metrics and progress                     |
|   - Wins and learnings                           |
|   - Q&A (the most important part)               |
+--------------------------------------------------+

THIRD WEEK: BOARD PREP (if applicable)
+--------------------------------------------------+
| Board meeting preparation                        |
|   - Financial package                            |
|   - Narrative update                             |
|   - Key decisions for board input                |
|   - Ask preparation (what you need from board)   |
+--------------------------------------------------+

FOURTH WEEK: STRATEGY
+--------------------------------------------------+
| Strategic thinking time                          |
|   - Is our strategy working?                     |
|   - What has changed in the market?              |
|   - What are we not seeing?                      |
|   - What should we start/stop/continue?          |
+--------------------------------------------------+
```

### Monthly All-Hands Design

```
ALL-HANDS MEETING (60 min)
+--------------------------------------------------+
| SEGMENT 1: STATE OF THE COMPANY (15 min)        |
| - Where we are (metrics, progress)              |
| - What is working and what is not               |
| - Transparent and honest                        |
|                                                  |
| SEGMENT 2: STRATEGY UPDATE (10 min)            |
| - Where we are going (reinforce direction)      |
| - Why it matters (connect to mission)           |
| - What is changing (if anything)                |
|                                                  |
| SEGMENT 3: TEAM SPOTLIGHT (10 min)              |
| - One team presents their work                  |
| - Rotates monthly across departments            |
| - Builds cross-team understanding               |
|                                                  |
| SEGMENT 4: CEO Q&A (25 min)                    |
| - Open questions, no filters                    |
| - CEO answers honestly or says "I do not know"  |
| - Written questions accepted for anonymity       |
+--------------------------------------------------+
```

## The CEO's Quarterly Cadence

### Quarterly Rhythm

```
QUARTER END (Last 2 weeks):
+--------------------------------------------------+
| WEEK 1: REVIEW                                   |
| - Score all OKRs (0.0-1.0)                      |
| - Revenue/financial close                        |
| - Customer cohort analysis                       |
| - Competitive landscape update                   |
| - Team health survey                             |
|                                                  |
| WEEK 2: PLANNING                                 |
| - Set next quarter OKRs                          |
| - Resource reallocation                          |
| - Budget adjustments                             |
| - Hiring plan updates                            |
| - Big rocks for next quarter (2-3 max)          |
+--------------------------------------------------+
```

### Quarterly Board Meeting

```
BOARD MEETING (3-4 hours, quarterly)
+--------------------------------------------------+
| SEGMENT 1: FINANCIAL REVIEW (45 min)            |
| - P&L, cash flow, balance sheet                 |
| - Key financial metrics vs. plan                |
| - Runway and funding needs                      |
|                                                  |
| SEGMENT 2: BUSINESS UPDATE (45 min)            |
| - Product: what shipped, what learned           |
| - Customers: growth, retention, NPS             |
| - Team: hires, departures, culture              |
| - Market: competitive moves, trends             |
|                                                  |
| SEGMENT 3: STRATEGIC DISCUSSION (60 min)        |
| - One or two strategic topics for board input   |
| - Not status updates -- real discussion         |
| - CEO asks for specific input/decisions         |
|                                                  |
| SEGMENT 4: CLOSED SESSION (30 min)              |
| - CEO performance feedback                      |
| - Sensitive topics (personnel, legal)           |
| - Board-only discussion                         |
+--------------------------------------------------+
```

## The CEO's Annual Cadence

### Annual Rhythm

```
Q4 (October-December): ANNUAL PLANNING
+--------------------------------------------------+
| October: Context setting and strategic review    |
| November: Objective setting and budgeting        |
| December: Plan finalization and communication    |
+--------------------------------------------------+

Q1 (January-March): EXECUTION LAUNCH
+--------------------------------------------------+
| January: Annual kickoff, team alignment          |
| February: Execution begins, early course-correct |
| March: Q1 close and first quarterly review       |
+--------------------------------------------------+

Q2 (April-June): MID-YEAR EXECUTION
+--------------------------------------------------+
| April: Q2 launch, momentum building             |
| May: Mid-year strategic check                    |
| June: Q2 close, mid-year course correction       |
+--------------------------------------------------+

Q3 (July-September): PREPARATION
+--------------------------------------------------+
| July: Q3 launch, execution focus                 |
| August: Pre-planning for next year               |
| September: Q3 close, annual planning begins      |
+--------------------------------------------------+
```

## Meeting Design Principles

### Meeting Hygiene

```
MEETING RULES (Non-Negotiable)
1. Every meeting has a PURPOSE (inform, decide, discuss, or align)
2. Every meeting has an AGENDA (sent 24+ hours in advance)
3. Every meeting has a DECISION-MAKER (not a committee)
4. Every meeting produces ACTIONS (owner + deadline)
5. Every meeting has a TIME LIMIT (and ends on time)
6. Meetings start on time regardless of who is present
7. No devices unless actively contributing
8. Silence is disagreement -- if you do not speak up, you agree
```

### Meeting Types

| Type | Purpose | Duration | Frequency | Attendees |
|------|---------|----------|-----------|-----------|
| 1:1 | Coaching, alignment | 30-60 min | Weekly | CEO + report |
| Standup | Status, blockers | 15 min | Daily | Team |
| Business review | Metrics, red flags | 60 min | Weekly | Leadership |
| All-hands | Alignment, transparency | 60 min | Monthly | Company |
| Board meeting | Governance, strategy | 3-4 hours | Quarterly | Board |
| Strategy offsite | Deep strategic thinking | 1-2 days | Annually | Leadership |
| Retrospective | Learning | 30-60 min | Per project | Project team |

### The Meeting Audit

Every quarter, audit all recurring meetings:

```
MEETING AUDIT
+--------------------------------------------------+
| Meeting: [Name]                                  |
| Frequency: [How often]                           |
| Duration: [How long]                             |
| Attendees: [Who attends]                         |
|                                                  |
| QUESTIONS:                                       |
| 1. Is this meeting still necessary?              |
| 2. Is the right person attending?                |
| 3. Could it be shorter?                          |
| 4. Could it be less frequent?                    |
| 5. Could it be an async update instead?          |
|                                                  |
| ACTION: [Keep / Modify / Cancel]                 |
+--------------------------------------------------+
```

## Cadence Anti-Patterns

| Anti-Pattern | Symptom | Fix |
|-------------|---------|-----|
| Meeting overload | CEO spends 80%+ time in meetings | Audit and cull; batch meetings on 2-3 days |
| No deep work | CEO never has 3+ hours of uninterrupted time | Block mornings; protect the calendar |
| Skipping reviews | Metrics review gets "too busy" to do | Make it the meeting that is never cancelled |
| Reactive cadence | CEO day is driven by inbox and Slack | Set the agenda the night before |
| All-hands theater | All-hands is a presentation, not dialogue | 40%+ of time must be Q&A |
| Board management vs. board partnership | CEO manages board instead of leveraging it | Bring real strategic questions |

---

**Operating cadence is the CEO's most powerful tool for organizational
discipline. It ensures that the right conversations happen at the right
time, that metrics are reviewed before they become crises, that strategy
is communicated relentlessly, and that the organization operates with
rhythm rather than chaos.**
