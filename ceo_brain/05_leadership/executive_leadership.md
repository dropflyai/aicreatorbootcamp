# Executive Leadership -- CEO Competencies and Leadership Models

## Overview

Executive leadership is the practice of aligning human energy toward
organizational outcomes. It is distinct from management (which ensures
efficient operation of existing systems) in that leadership creates the
conditions for exceptional performance, navigates ambiguity, and transforms
organizations.

This module codifies the competencies, models, and practices that define
effective CEO-level leadership, drawing on Drucker, Greenleaf, Hersey and
Blanchard, and contemporary research from HBS and Stanford GSB.

## The Four CEO Competencies

### Competency 1: Vision

Vision is the ability to see a future state that others cannot yet perceive
and to articulate it so compellingly that people choose to work toward it.

Components of CEO vision:
1. **Pattern recognition** -- seeing connections between disparate signals
2. **Future projection** -- extrapolating trends to logical conclusions
3. **Narrative construction** -- translating insight into story
4. **Conviction** -- maintaining commitment when evidence is ambiguous

```
VISION DEVELOPMENT PROCESS
1. Immerse in customer reality (100+ conversations)
2. Study adjacent industries for patterns
3. Identify the unsustainable current state
4. Project the inevitable future state
5. Articulate the bridge (your company)
6. Test with skeptics (not sycophants)
7. Refine until it is clear and compelling
```

Vision failure modes:
- **Hallucination** -- vision disconnected from reality
- **Incrementalism** -- vision too modest to inspire
- **Solipsism** -- vision that excites the CEO but no one else
- **Rigidity** -- vision that cannot adapt to new information

### Competency 2: Talent Density

Netflix's concept (Reed Hastings, *No Rules Rules*, 2020): the density of
high-performing talent in the organization determines its capability ceiling.

CEO responsibility for talent:
1. **Define the bar** -- What does "excellent" look like for each role?
2. **Attract** -- Why would A-players choose your company?
3. **Select** -- How do you identify A-players in the hiring process?
4. **Develop** -- How do you grow A-players into leaders?
5. **Retain** -- What keeps A-players engaged and committed?
6. **Remove** -- How do you exit people who do not meet the bar?

```
TALENT DENSITY FRAMEWORK
+--------------------------------------------+
| A-Players (top 10%)                        |
|   - Self-directed, high output             |
|   - Raise the bar for everyone             |
|   - Attract other A-players                |
|   ACTION: Retain, develop, reward          |
+--------------------------------------------+
| B-Players (middle 70%)                     |
|   - Solid performers, reliable             |
|   - With coaching, some become A-players   |
|   - Without coaching, some become C-players|
|   ACTION: Coach, develop, set expectations |
+--------------------------------------------+
| C-Players (bottom 20%)                     |
|   - Below standard performance             |
|   - Drag on team morale and output         |
|   - Consume disproportionate management    |
|   ACTION: Coach briefly; exit if no improve|
+--------------------------------------------+
```

The uncomfortable truth: keeping C-players is not kindness -- it is
cruelty to the A-players who have to compensate, to the B-players who
learn that mediocrity is tolerated, and to the C-players who would
perform better in a role that matches their strengths.

### Competency 3: Culture

Culture is the set of behaviors, norms, and values that define how
an organization operates when no one is watching. The CEO is the chief
culture officer whether they want to be or not.

Culture levers the CEO controls:
1. **Who gets hired** (culture add, not just culture fit)
2. **Who gets fired** (especially for values violations)
3. **Who gets promoted** (the strongest culture signal)
4. **What gets rewarded** (incentives shape behavior)
5. **What the CEO tolerates** (what you permit, you promote)
6. **What the CEO models** (behavior, not words, defines culture)

### Competency 4: Communication

The CEO is the chief storyteller, the chief meaning-maker, and the
chief communicator. Communication is not a subset of the CEO job --
it IS the CEO job.

```
CEO COMMUNICATION MATRIX
+--------------------+---------------------------+
| Audience           | Communication Objective   |
+--------------------+---------------------------+
| Employees          | Alignment, motivation     |
| Board              | Governance, support       |
| Investors          | Confidence, transparency  |
| Customers          | Trust, value              |
| Partners           | Collaboration, commitment |
| Press/Public       | Reputation, narrative     |
| Candidates         | Attraction, excitement    |
+--------------------+---------------------------+
```

## Servant Leadership (Robert Greenleaf, 1970)

### The Concept

The servant-leader is servant first. The best test: "Do those served grow
as persons? Do they, while being served, become healthier, wiser, freer,
more autonomous, more likely themselves to become servants?"

### CEO Application

The servant-leader CEO inverts the traditional hierarchy:

```
TRADITIONAL:                 SERVANT LEADERSHIP:
     CEO                         Customers
      |                             |
    VPs                          Front-line
      |                             |
  Directors                      Managers
      |                             |
  Managers                       Directors
      |                             |
  Front-line                       VPs
      |                             |
  Customers                        CEO
```

In servant leadership, the CEO's job is to remove obstacles for the
people who create value for customers. The CEO serves the team so the
team can serve the customer.

Practical servant leadership behaviors:
1. **Ask "What do you need from me?"** -- not "What have you done for me?"
2. **Remove blockers** -- the CEO's power is the ability to clear paths
3. **Provide context** -- share information so people can decide well
4. **Develop people** -- invest in your team's growth
5. **Take blame, give credit** -- always

## Situational Leadership (Hersey and Blanchard)

### The Model

No single leadership style works for all situations. Effective leaders
match their style to the follower's development level:

```
HIGH SUPPORT
      |
  S3: SUPPORTING     S2: COACHING
  (High support,     (High support,
   Low direction)     High direction)
      |                    |
------+--------------------+------> HIGH DIRECTION
      |                    |
  S4: DELEGATING     S1: DIRECTING
  (Low support,      (Low support,
   Low direction)     High direction)
      |
LOW SUPPORT
```

### Development Levels

| Level | Description | Leadership Style |
|-------|------------|-----------------|
| D1: Enthusiastic beginner | High commitment, low competence | S1: Directing (tell them what and how) |
| D2: Disillusioned learner | Lower commitment, growing competence | S2: Coaching (explain why, encourage) |
| D3: Capable but cautious | Variable commitment, high competence | S3: Supporting (listen, facilitate) |
| D4: Self-reliant achiever | High commitment, high competence | S4: Delegating (empower, trust) |

### CEO Application

The CEO applies situational leadership to direct reports:
- **New VP** (first 90 days): S2 Coaching -- frequent 1:1s, context sharing,
  explicit expectations, close support
- **Established VP** (performing well): S4 Delegating -- set goals, review
  results, intervene only on exceptions
- **VP in new challenge** (e.g., scaling for first time): S3 Supporting --
  available, encouraging, asking questions not giving answers
- **VP struggling** (not meeting expectations): S1 Directing -- explicit
  expectations, close monitoring, specific improvement plan

## The CEO as Coach

### Why Coaching Matters

The CEO who only directs creates a dependent organization. The CEO who
coaches creates an organization of leaders who can operate independently.

Michael Bungay Stanier (*The Coaching Habit*, 2016) identifies seven
essential coaching questions:

1. **"What is on your mind?"** -- Opens the conversation
2. **"And what else?"** -- Goes deeper (the AWE question)
3. **"What is the real challenge here for you?"** -- Focuses on the real issue
4. **"What do you want?"** -- Clarifies the desired outcome
5. **"How can I help?"** -- Offers support without taking over
6. **"If you are saying yes to this, what are you saying no to?"** -- Forces tradeoffs
7. **"What was most useful for you?"** -- Creates learning

### CEO Coaching Practice

In every 1:1, the CEO should spend at least 25% of the time coaching:
- Listen more than talk (aim for 70% listening)
- Ask questions before giving answers
- Let the direct report solve their own problems when possible
- Provide perspective and context (the CEO's unique value)
- Give both positive and constructive feedback

## Executive Presence

### What It Is

Executive presence is the ability to command a room, inspire confidence,
and project authority. Research from the Center for Talent Innovation
identifies three components:

```
EXECUTIVE PRESENCE
+--------------------------------------------+
| GRAVITAS (67% of executive presence)       |
|   - Confidence and decisiveness            |
|   - Emotional intelligence                 |
|   - Reputation and track record            |
|   - Vision and strategic thinking          |
|   - Composure under pressure               |
+--------------------------------------------+
| COMMUNICATION (28%)                        |
|   - Speaking with authority                |
|   - Active listening                       |
|   - Commanding a room                      |
|   - Storytelling ability                   |
|   - Reading an audience                    |
+--------------------------------------------+
| APPEARANCE (5%)                            |
|   - Appropriate and polished               |
|   - Consistent with role and context       |
|   - Projecting energy and health           |
+--------------------------------------------+
```

### Building Executive Presence

1. **Preparation** -- presence comes from mastery of the material
2. **Composure** -- practice staying calm under pressure
3. **Clarity** -- say less, mean more; avoid hedging language
4. **Conviction** -- state your view clearly before inviting debate
5. **Vulnerability** -- admitting what you do not know builds trust
6. **Listening** -- presence is not just about speaking; it is about
   the quality of your attention

## Leadership in the Brain System

### The CEO Brain's Leadership Role

The CEO Brain exercises leadership through:

1. **Setting direction** -- defining vision, strategy, and priorities
   that all brains follow
2. **Making the hard calls** -- resolving conflicts, killing projects,
   prioritizing when resources are scarce
3. **Modeling standards** -- the quality of CEO Brain output sets the
   bar for all brains
4. **Developing brains** -- as new brains are built, the CEO Brain
   defines their scope, quality bar, and interfaces
5. **Protecting culture** -- ensuring brain interactions are collaborative,
   honest, and outcome-oriented

### Adaptive Leadership for Different Contexts

| Context | Leadership Style | Primary Focus |
|---------|-----------------|---------------|
| Building (startup) | Visionary + directing | Set direction, inspire |
| Scaling | Coaching + delegating | Develop leaders, build systems |
| Crisis | Directing + commanding | Stabilize, decide quickly |
| Strategy | Collaborative + facilitating | Synthesize perspectives |
| Operations | Delegating + monitoring | Trust systems, review output |

## CEO Leadership Anti-Patterns

| Anti-Pattern | Description | Fix |
|-------------|------------|-----|
| The Genius CEO | Believes they are the smartest person; does not listen | Hire people smarter than you; listen |
| The Absent CEO | Delegates everything; never present | Be present at critical moments |
| The Micromanager | Cannot let go of details | Define outcomes, not methods |
| The People-Pleaser | Avoids conflict; does not fire | Courage is a leadership requirement |
| The Visionary Only | Great at vision, terrible at execution | Build execution systems; hire COO |
| The Operator Only | Great at execution, no vision | Develop vision or hire visionary leader |

---

**Executive leadership is not a title -- it is a practice. The CEO Brain
exercises leadership through every decision, delegation, and communication.
Vision without execution fails. Execution without vision is pointless.
The CEO Brain provides both.**
