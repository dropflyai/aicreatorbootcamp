# Culture -- Culture as Strategy

## Overview

Culture is the set of shared behaviors, norms, values, and assumptions that
define how an organization operates. It is not what you say on the website;
it is what happens when no one is looking. Peter Drucker (attributed):
"Culture eats strategy for breakfast." More precisely: culture IS strategy,
expressed through daily behavior.

The CEO is the chief culture officer whether they accept the role or not.
Every action, every inaction, every hire, every fire, every promotion, and
every toleration shapes culture. This module codifies how to design, build,
reinforce, and evolve organizational culture deliberately.

## Culture as a Strategic Asset

### Why Culture Matters

Research consistently demonstrates that culture is a primary driver of
organizational performance:

1. **Execution speed** -- high-trust cultures make decisions faster
   because less time is spent on politics, posturing, and CYA
2. **Talent attraction** -- A-players choose companies based on culture
   as much as compensation
3. **Talent retention** -- the #1 reason people leave is culture/manager,
   not compensation
4. **Innovation** -- psychological safety enables risk-taking and
   experimentation
5. **Resilience** -- strong cultures survive crises that destroy weak ones
6. **Competitive advantage** -- culture is the one asset competitors
   cannot copy

        (Collaborate)   |        (Create)

Most organizations are a blend, with one or two dominant types. The CEO

| Innovation-driven | Adhocracy (create) |
| Customer-intimacy | Clan (collaborate) |
| Operational excellence | Hierarchy (control) |
| Market leadership | Market (compete) |

## Values: From Wall to Behavior

### The Values Problem

Most company values are meaningless. They are aspirational platitudes
that do not differentiate one company from another: "integrity,"
"excellence," "teamwork," "innovation." Every company claims these
values; few live them.

Patrick Lencioni (*The Advantage*, 2012) distinguishes four types:

```
+--------------------+----------------------------------+
| Core Values        | Deeply held, non-negotiable.     |
|                    | You would keep them even if they |
|                    | became a competitive disadvantage|
+--------------------+----------------------------------+
| Aspirational Values| What you want to become but are  |
|                    | not yet. Gap between stated and  |
|                    | actual behavior.                 |
+--------------------+----------------------------------+
| Permission-to-Play | Minimum behavioral standards.    |
| Values             | Expected of everyone (honesty,   |
|                    | respect). Not differentiating.   |
+--------------------+----------------------------------+
| Accidental Values  | Values that arise organically    |
|                    | without intentional cultivation. |
|                    | May be positive or negative.     |
+--------------------+----------------------------------+
```

### Making Values Operational

A value is operational only when it:
1. Has specific, observable behaviors associated with it
2. Is used in hiring decisions (screen for it)
3. Is used in firing decisions (fire for violating it)
4. Is used in promotion decisions (promote exemplars)
5. Is reinforced through stories, rituals, and recognition
6. Is modeled by leadership (especially the CEO)

```
VALUE OPERATIONALIZATION TEMPLATE
+--------------------------------------------------+
| VALUE: [Name]                                    |
|                                                  |
| DEFINITION: [What it means in 1-2 sentences]    |
|                                                  |
| BEHAVIORS THAT DEMONSTRATE IT:                   |
| - [Observable behavior 1]                       |
| - [Observable behavior 2]                       |
| - [Observable behavior 3]                       |
|                                                  |
| BEHAVIORS THAT VIOLATE IT:                       |
| - [Observable violation 1]                      |
| - [Observable violation 2]                      |
|                                                  |
| HIRING QUESTION:                                 |
| "Tell me about a time when [scenario that tests |
|  this value]"                                    |
|                                                  |
| RECOGNITION:                                     |
| [How we celebrate this value in practice]        |
+--------------------------------------------------+
```

## Culture Reinforcement Mechanisms

### The Culture Flywheel

```
     Values defined
          |
          v
    Hiring for values -----> Values-aligned team
          ^                       |
          |                       v
    Stories reinforce <----- Behaviors observed
    values                       |
          ^                       v
          |               Promotions reward
          +---------- values exemplars
```

### Mechanism 1: Hiring

Culture is built one hire at a time. Every person you add either
strengthens or dilutes the culture.

Interview for values:
- Behavioral questions: "Tell me about a time when..."
- Scenario questions: "What would you do if..."
- Reference checks: "How does this person handle [value-relevant scenario]?"
- Culture interviews: dedicated interview focused solely on values alignment

The "no asshole rule" (Bob Sutton): A brilliant person who damages culture
is a net negative. Technical skill does not compensate for cultural toxicity.

### Mechanism 2: Firing

What you tolerate defines your culture more than what you celebrate.

Fire for values violations:
- A top performer who undermines teammates must go
- A brilliant engineer who creates a toxic environment must go
- A rainmaker salesperson who lies to customers must go

These are the hardest and most important cultural decisions. The team is
watching. When you fire the top performer for values violations, you send
the most powerful cultural signal possible: "We mean what we say."

### Mechanism 3: Promotion

Promotion is the ultimate culture amplifier. Who gets promoted tells
the organization what is actually valued.

```
PROMOTION CRITERIA:
1. Performance (results) -- necessary but not sufficient
2. Values alignment (behavior) -- mandatory, non-negotiable
3. Leadership capability (potential) -- for management roles
4. Impact on others (multiplier effect) -- raises the team's output

NEVER promote someone who delivers results but violates values.
The short-term gain is not worth the long-term cultural damage.
```

### Mechanism 4: Stories and Rituals

Culture is transmitted through stories, not documents. The CEO is the
chief storyteller.

Types of cultural stories:
- **Origin stories** -- how the company started, what drove the founders
- **Hero stories** -- people who exemplified values in difficult situations
- **Cautionary stories** -- what happens when values are violated
- **Customer stories** -- how values translate to customer impact

Cultural rituals:
- **Weekly wins** -- celebrating value-aligned achievements publicly
- **Onboarding** -- new hire culture immersion in the first week
- **Annual awards** -- named for values, not just performance
- **All-hands recognition** -- CEO personally highlights value exemplars

## Dunbar Transitions: Culture at Scale

### Robin Dunbar's Social Layers

Anthropologist Robin Dunbar identified natural group size limits for human
social organization. Each transition requires deliberate cultural
intervention:

```
DUNBAR NUMBER: 150 (max stable social relationships)

5 people:     Intimate group (co-founders)
15 people:    Sympathy group (core team)
50 people:    Close group (early company)
150 people:   Dunbar number (natural cultural limit)
500 people:   Acquaintance group (requires formal culture)
1500 people:  Tribal group (requires cultural infrastructure)
```

### Cultural Transitions at Each Stage

#### 5 to 15 People (Founders to Core Team)

Culture is implicit. Everyone knows everyone. Values are transmitted by
proximity and osmosis. The CEO models culture directly.

Actions needed:
- Articulate values explicitly (even if informally)
- Hire slowly and deliberately for cultural alignment
- Create shared experiences that bond the team

#### 15 to 50 People (Core Team to Early Company)

Culture begins to dilute. New hires do not have direct CEO contact.
Subcultures begin to form.

Actions needed:
- Write down values and behaviors explicitly
- Implement cultural onboarding for every new hire
- CEO spends significant time on culture (all-hands, skip-levels)
- Begin cultural rituals (weekly wins, etc.)

#### 50 to 150 People (The Critical Transition)

This is the most dangerous cultural transition. The CEO can no longer
know everyone personally. Middle management emerges. Values must be
transmitted through systems, not just personal relationships.

Actions needed:
- Formalize culture documentation
- Train managers on cultural expectations
- Implement values-based performance reviews
- Create cultural ambassadors program
- CEO communicates culture in every all-hands

#### 150 to 500 People (Scaling Culture)

Beyond Dunbar's number, culture cannot be maintained through personal
relationships. It must be encoded in systems, processes, and artifacts.

Actions needed:
- Culture team or dedicated People function
- Formal cultural onboarding program (multi-day)
- Values-based hiring criteria in every job description
- Cultural metrics (engagement surveys, eNPS)
- CEO culture reinforcement at scale (written comms, video)

#### 500+ People (Organizational Culture)

At this scale, subcultures are inevitable and healthy. The CEO's job
shifts from maintaining one culture to ensuring that all subcultures
align with core values while allowing functional variation.

Actions needed:
- Core values remain constant; functional norms can vary
- Cultural alignment in M&A (culture due diligence)
- Regular cultural health assessments
- CEO as cultural symbol and final arbiter

## Cultural Debt

### The Concept

Like technical debt, cultural debt accumulates when you make cultural
compromises for short-term gain:

```
CULTURAL DEBT EXAMPLES:
- Hiring for skills without screening for values
- Tolerating toxic behavior from high performers
- Promoting the wrong people for the wrong reasons
- Not addressing interpersonal conflict
- Allowing communication norms to degrade
- Ignoring diversity and inclusion
- Not investing in management development
```

### Cultural Debt Compounds

```
Year 1: Hire one person who does not share values
Year 2: They hire two more like them
Year 3: A subculture forms that conflicts with stated values
Year 4: The original culture bearers start leaving
Year 5: The stated values and actual culture are completely divergent
```

### Paying Down Cultural Debt

1. **Audit** -- honestly assess where stated values and actual behavior
   diverge
2. **Prioritize** -- which cultural debts are most damaging?
3. **Address the people issues** -- the hardest and most important step.
   People who persistently violate values must change or leave.
4. **Reinforce the target culture** -- increase rituals, stories,
   recognition that amplify desired behaviors
5. **Measure** -- track cultural health metrics over time

## Building Culture in the Brain System

### Brain System Cultural Values

The brain system operates on these cultural principles:

1. **Specialist excellence** -- each brain maintains the highest standard
   in its domain
2. **Collaborative integration** -- brains work together, not in silos
3. **Intellectual honesty** -- brains report reality, not what the CEO
   Brain wants to hear
4. **Outcome orientation** -- measured by output, not activity
5. **Continuous improvement** -- every project produces learning
6. **Transparent tradeoffs** -- all decisions include explicit tradeoffs

### Cultural Enforcement in Brain Interactions

- Quality gates enforce excellence
- Handoff protocols enforce collaboration
- Memory logging enforces learning
- Decision documentation enforces transparency
- Retrospectives enforce improvement
