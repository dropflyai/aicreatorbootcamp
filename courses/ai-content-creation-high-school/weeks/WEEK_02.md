# Week 2: "The Art of the Prompt"

## Theme: Words That Control AI

---

## Learning Objectives

By the end of this week, students will be able to:
1. Master the ROLE + CONTEXT + TASK + CONSTRAINTS prompt structure
2. Iterate and refine prompts for better outputs
3. Write hooks that stop the scroll (first 3 seconds)
4. Generate platform-optimized captions
5. Recognize and fix low-quality prompt patterns

---

## Session 1: Prompt Battle (90 minutes)

### Opening Hook (15 min)
**"Prompt Battle"**

Show two AI outputs side by side:

**Bad Prompt:** "Write me a TikTok caption"
**Output:** Generic, forgettable caption

**Good Prompt:** "You're a Gen Z content creator with 500K followers known for sarcastic humor. Write a TikTok caption for a video about the struggle of waking up for school. Use slang naturally, include one unexpected twist, and end with a question to drive comments. Max 100 characters."
**Output:** Engaging, specific, platform-perfect caption

**Class Vote:** Which would you engage with?

**The Lesson:** The quality of your output is directly tied to the quality of your input.

### Core Content: Prompt Anatomy (30 min)

**The RCTC Framework:**

```
R - ROLE: Who should the AI be?
C - CONTEXT: What's the situation/background?
T - TASK: What specifically do you want?
C - CONSTRAINTS: What are the limits/requirements?
```

**Examples:**

| Component | Bad | Good |
|-----------|-----|------|
| ROLE | (none) | "You're a viral TikTok creator who specializes in educational content" |
| CONTEXT | (none) | "I'm a high school junior making content about study tips" |
| TASK | "Write a caption" | "Write 3 hook options for a video about the Pomodoro technique" |
| CONSTRAINTS | (none) | "Each under 10 words, include an emoji, make one a question" |

**Live Demo:** Transform bad prompts into good ones in real-time

### Activity: Prompt Transformation (35 min)

**Round 1: Fix These Prompts (15 min)**

Students receive 5 bad prompts and must improve them:

1. "Write something funny"
2. "Help me with my video"
3. "Give me ideas"
4. "Make this better" + [text]
5. "Write a script"

**Round 2: Prompt Battle Tournament (20 min)**

- Pair students up
- Both write a prompt for the same task
- Class votes on which output is better
- Winners advance

**Tasks:**
- Write a hook for a "day in my life" video
- Create a caption for a gym transformation post
- Write an intro for a podcast about gaming

### Wrap-Up (10 min)
- Key takeaway: Specific > Vague, always
- Preview Session 2: Platform-specific writing
- Homework: Collect 5 captions you love from creators you follow

## Session 2: Hooks and Captions (90 minutes)

### Quick Review (10 min)
- RCTC framework recap
- Share favorite captions from homework
- What made them work?

### Core Content: The Hook (25 min)

**The 0.5-Second Rule:**
You have half a second to stop someone from scrolling. That's it.

**Hook Formulas That Work:**

| Formula | Example | Why It Works |
|---------|---------|--------------|
| **Bold Statement** | "Most people study completely wrong" | Challenges assumptions |
| **Question** | "Would you skip class if you knew this?" | Creates curiosity |
| **Number + Outcome** | "3 apps that got me into college" | Specific, promises value |
| **POV** | "POV: You finally understand math" | Relatable scenario |
| **Shock** | "I got rejected from 12 colleges. Here's what happened next." | Emotional pull |
| **Direct Address** | "Stop scrolling if you have a test tomorrow" | Personal connection |

**The Hook Test:**
- Does it create a curiosity gap?
- Would YOU stop scrolling?
- Does it promise something specific?

### Core Content: Platform-Specific Writing (20 min)

**TikTok:**
- Casual, conversational
- Slang is expected
- Emojis: moderate use
- Hashtags: 3-5 relevant ones
- Length: Short and punchy

**Instagram:**
- Slightly more polished
- Can be longer (first line is hook)
- Emojis: common
- Hashtags: in caption or first comment
- Save-worthy content performs best

**YouTube:**
- More informational tone
- Description matters for SEO
- Keywords in first 2 sentences
- Timestamps appreciated
- Drives to subscribe

**Twitter/X:**
- Punchy, quotable
- Thread-friendly
- Hot takes work
- Conversation starters
- Character limit forces clarity

### Activity: 5 Platform Captions (30 min)

**The Challenge:**
Take ONE content idea and write captions for 5 platforms.

**Content Idea Example:** "The best study technique nobody uses"

Students write:
1. TikTok caption (casual, hook-first)
2. Instagram caption (hook + value + CTA)
3. YouTube description (SEO + timestamps)
4. Twitter/X post (punchy, shareable)
5. LinkedIn post (professional angle)

**Prompt Template:**
```
I'm creating content about [TOPIC]. Write a [PLATFORM] caption that:
- Hooks in the first line
- Matches the platform's culture
- Includes a call-to-action
- Is appropriate for a high school audience

Platform culture notes: [TikTok = casual/slang, Instagram = aesthetic/saves, etc.]
```

### Peer Review (10 min)
- Swap with partner
- Score each caption: Would you stop scrolling? (1-5)
- Give one specific improvement suggestion

### Wrap-Up (5 min)
- Best captions shared with class
- Weekly project reminder

## Prompt Templates for Students

### Idea Generation
```
You're a content strategist for Gen Z creators. I create content about [NICHE] for [PLATFORM].

Generate 10 specific video ideas that:
- Are trending or evergreen
- Match my niche
- Can be filmed with just a phone
- Would appeal to [TARGET AUDIENCE]

For each idea, include: Title, Hook (first line), and Why it could go viral.
```

### Script Writing
```
You're a [PLATFORM] scriptwriter known for [STYLE: funny/educational/dramatic].

Write a [LENGTH] second script about [TOPIC].

Structure:
- Hook (first 3 seconds): Stop the scroll
- Body: Deliver the value/entertainment
- CTA: What should viewers do?

Style notes: [Casual/formal, humor level, energy level]
My voice: [Describe your style]
```

### Caption Writing
```
Write a [PLATFORM] caption for my post about [TOPIC].

Requirements:
- Hook in first line (stop the scroll)
- [Platform-specific style]
- Include [emoji/hashtag requirements]
- End with [CTA type: question/action/follow]
- Max [X] characters

My brand voice: [Describe]
```

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Prompts still too vague | Have them add one more constraint each time |
| All captions sound the same | Show platform-native examples side by side |
| Can't think of content idea | Provide list of 20 niche options |
| Prompt Battle ties | Use "which would get more comments?" as tiebreaker |
