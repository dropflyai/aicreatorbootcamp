# Content Performance — Video Content Audit and Competitive Analysis

## Performance Intelligence for Video Strategy

Content performance analysis transforms raw video analytics into strategic intelligence. It answers critical questions: Which content is working and why? Which content should be discontinued? How does our performance compare to competitors? What content gaps exist? Without systematic performance analysis, content strategy is driven by intuition rather than evidence — resulting in repeated investment in underperforming formats and missed opportunities in high-potential areas.

---

## 1. Video Content Audit

### Why Audit Video Content?

A video content audit is a systematic evaluation of all published video content against performance metrics, strategic alignment, and production efficiency. Regular audits prevent three common failures:

1. **Content bloat:** Continuing to produce content types that do not perform
2. **Strategic drift:** Content that no longer aligns with current business objectives
3. **Missed patterns:** Failing to recognize what is working and doubling down

### Audit Framework

**Step 1: Inventory**
Catalog all published video content with metadata:

| Field | Description |
|-------|-------------|
| Video ID | Unique identifier |
| Title | Video title |
| Platform | Where published (YouTube, TikTok, LinkedIn, etc.) |
| Publish date | Date published |
| Content type | Tutorial, testimonial, brand, product demo, etc. |
| Topic/category | Subject matter category |
| Length | Duration |
| Production tier | DIY, semi-pro, professional, premium |
| Production cost | Total cost including internal time |
| Funnel stage | Awareness, consideration, decision, retention |
| Status | Active, underperforming, candidate for update, candidate for removal |

**Step 2: Performance Data Collection**
For each video, collect platform-specific metrics:

| Metric Category | Specific Metrics |
|-----------------|-----------------|
| Reach | Views, impressions, unique viewers |
| Engagement | AVD, completion rate, likes, comments, shares, saves |
| Conversion | CTR, leads generated, sign-ups, revenue attributed |
| SEO | Search impressions, search CTR, ranking keywords |
| Growth | Subscribers/followers gained |

**Step 3: Performance Classification**

Score each video on a composite performance index, then classify:

```
Performance Index = (Normalized Views * 0.25) +
                    (Normalized Engagement * 0.25) +
                    (Normalized Conversion * 0.30) +
                    (Normalized AVD * 0.20)
```

| Classification | Criteria | Action |
|---------------|---------|--------|
| Star performer | Top 10% by performance index | Analyze success factors; create similar content |
| Solid performer | Top 25% (excluding stars) | Maintain; optimize incrementally |
| Average | Middle 50% | Evaluate cost vs. value; optimize or discontinue |
| Underperformer | Bottom 25% | Diagnose failure cause; discontinue or dramatically revise |
| Zombie content | Old content with minimal ongoing views | Decide: update, redirect, or remove |

**Step 4: Pattern Analysis**

Analyze across the classified content to identify patterns:

| Pattern Dimension | Analysis Questions |
|-------------------|-------------------|
| Content type | Which types (tutorial, testimonial, etc.) perform best? |
| Topic | Which topics drive the most views, engagement, and conversion? |
| Length | Is there an optimal length for our audience? |
| Format | Do certain formats (talking head, screen share, animation) outperform? |
| Production tier | Does higher production value correlate with better performance? |
| Publish timing | Does day of week or time of day affect performance? |
| Thumbnail style | Which thumbnail styles drive higher CTR? |
| Title patterns | Which title formulas drive higher CTR? |
| Funnel stage | Where in the funnel does video contribute most? |

## 2. Evergreen vs. Topical Content Strategy

### Content Lifespan Categories

```
┌───────────────────────────────────────────────────────────┐
│                   CONTENT LIFESPAN MAP                     │
├─────────────────────┬─────────────────────────────────────┤
│   EVERGREEN          │   TOPICAL                           │
│   (Long lifespan)    │   (Short lifespan)                  │
├─────────────────────┼─────────────────────────────────────┤
│ How-to tutorials     │ News commentary                     │
│ Foundational guides  │ Trend reactions                     │
│ FAQ/common questions │ Event coverage                      │
│ Product overviews    │ Seasonal content                    │
│ Customer stories     │ Product launch announcements        │
│ Concept explainers   │ Industry news analysis              │
│ Process walkthroughs │ Challenge/trending content          │
├─────────────────────┼─────────────────────────────────────┤
│ Lifespan: 1-5+ years│ Lifespan: 1 day - 3 months          │
│ Discovery: Search    │ Discovery: Feed/algorithm            │
│ Growth: Compound     │ Growth: Spike then decay             │
│ Investment: Higher   │ Investment: Lower per piece          │
│ ROI timeline: Slow   │ ROI timeline: Immediate or never     │
│ but compound         │                                     │
└─────────────────────┴─────────────────────────────────────┘
```

### Optimal Content Mix

| Channel Maturity | Evergreen % | Topical % | Rationale |
|-----------------|-------------|-----------|-----------|
| New channel (0-6 months) | 70-80% | 20-30% | Build search-discoverable library |
| Growing channel (6-18 months) | 50-60% | 40-50% | Balance discovery with feed freshness |
| Established channel (18+ months) | 40-50% | 50-60% | Existing library compounds; invest in timeliness |

### Evergreen Content Maintenance

Evergreen content is not "set and forget." It requires maintenance:

| Maintenance Action | Cadence | Trigger |
|-------------------|---------|---------|
| Accuracy review | Every 12 months | Scheduled |
| Performance check | Every 6 months | Scheduled |
| Refresh/re-record | When outdated or underperforming | Performance decline or factual obsolescence |
| SEO re-optimization | Every 12 months | Search ranking decline |
| Thumbnail refresh | When CTR declines below benchmark | CTR metric trigger |
| Add end screens to new content | Ongoing | New related content published |

## 3. Series Performance Analysis

### Why Series Matter

Video series build recurring viewership, subscribe intent, and algorithmic momentum. YouTube's algorithm heavily rewards channels that generate consistent viewership patterns.

### Series Performance Metrics

| Metric | Definition | Why It Matters |
|--------|-----------|---------------|
| Episode-to-episode retention | % of viewers who watch the next episode | Measures series stickiness |
| Series subscriber conversion | Subscribers gained per series episode vs. standalone | Measures franchise value |
| Average performance trajectory | Do later episodes perform better or worse? | Growth = audience building; decline = fatigue |
| Series vs. standalone | Performance of series episodes vs. one-off videos | Validates series investment |
| Binge factor | % of viewers who watch 3+ episodes in a session | Measures addictiveness |

### Series Health Assessment

| Status | Indicator | Action |
|--------|-----------|--------|
| Healthy growth | Each episode outperforms the last by 5-15% | Continue; invest more |
| Stable | Consistent performance across episodes | Maintain; test format variations |
| Early fatigue | Episodes 3-5 declining after strong start | Refresh format; shorten series arc |
| Terminal decline | 3+ consecutive episodes declining >15% | End series gracefully; analyze what changed |
| Breakthrough | Single episode dramatically outperforms | Analyze breakthrough; extract replicable elements |

### Series Planning Framework

| Element | Specification |
|---------|--------------|
| Series concept | [Core premise that can sustain multiple episodes] |
| Episode count | [Planned number: finite series (5-12) or ongoing] |
| Cadence | [Weekly, bi-weekly, monthly] |
| Format consistency | [Same structure each episode? Same length? Same host?] |
| Arc planning | [Does the series tell an overarching story or are episodes standalone?] |
| Success criteria | [Metrics that determine whether series continues] |
| Exit criteria | [Metrics that trigger series conclusion] |

## 4. Competitive Video Analysis

### Competitor Identification

Identify competitors in the video space (these may differ from business competitors):

| Category | Description |
|----------|-------------|
| Direct competitors | Companies selling similar products/services |
| Content competitors | Creators covering the same topics for the same audience |
| Aspirational benchmarks | Best-in-class video operations to learn from |
| Platform competitors | Anyone competing for the same audience's attention |

### Competitive Analysis Framework

**Quantitative Analysis:**

| Metric | Your Channel | Competitor A | Competitor B | Competitor C |
|--------|-------------|-------------|-------------|-------------|
| Total subscribers/followers | [#] | [#] | [#] | [#] |
| Avg views per video (last 30 days) | [#] | [#] | [#] | [#] |
| Upload frequency | [#/week] | [#/week] | [#/week] | [#/week] |
| Avg engagement rate | [%] | [%] | [%] | [%] |
| Estimated monthly views | [#] | [#] | [#] | [#] |
| Growth rate (subs/month) | [#] | [#] | [#] | [#] |

**Qualitative Analysis:**

|-----------|-------------|-------------|-------------|

| Production quality | [Low/Med/High] | [Level] | [Level] |

| CTA strategy | [Description] | [Description] | [Description] |
| Community engagement | [Low/Med/High] | [Level] | [Level] |
| Monetization model | [Description] | [Description] | [Description] |

### Competitive Gap Analysis

| Topic/Theme | Our Coverage | Competitor Coverage | Gap Type |
|-------------|-------------|--------------------|-----------|
| [Topic A] | Strong (10+ videos) | Moderate | Our advantage |
| [Topic B] | None | Strong (20+ videos) | Content gap (opportunity) |
| [Topic C] | Moderate | None | Emerging advantage |
| [Topic D] | None | None | White space (evaluate demand) |

### Competitor Content Reverse Engineering

For top-performing competitor videos, analyze:

1. **Title pattern:** What formula do they use? (How to, Number list, Question, etc.)
2. **Thumbnail pattern:** What visual elements recur in their highest-CTR thumbnails?
3. **Hook structure:** How do they open their videos? What makes viewers stay?
4. **Content structure:** How is the video organized? What pacing techniques do they use?
5. **CTA strategy:** How and when do they ask for subscribes, engagement, conversion?
6. **Comment engagement:** How do they interact with their community?

## 5. Performance Review Cadence

### Weekly
- New video performance vs. channel average
- Trending content identification
- Engagement response (comment replies)
- Short-form content performance

### Monthly
- Full channel analytics review
- Content type performance comparison
- Series health check
- Competitive snapshot

### Quarterly
- Content audit (full inventory review)
- Strategic alignment assessment
- ROI analysis and budget review
- Content strategy adjustment

### Annually
- Comprehensive competitive analysis
- Content strategy overhaul
- Production capability assessment
- Technology and tool review
